# 🧅 Smart Onion Storage System

## 📌 Overview
The Smart Onion Storage System is an IoT-based solution aimed at minimizing 30–40% post-harvest losses by maintaining optimal environmental conditions within storage units. It enhances the shelf life of onions through continuous monitoring and intelligent control.

## 🚀 Features
- Real-time monitoring of temperature, humidity, and gas levels  
- Integration of sensors with Arduino/ESP32 for efficient data processing  
- Automated regulation of storage conditions  
- Cloud-based data visualization using ThingSpeak  

## 🛠️ Technologies Used
- Arduino / ESP32  
- DHT11 Sensor  
- Gas Sensor  
- ThingSpeak Cloud Platform  

## ⚙️ Working
The system utilizes environmental sensors to collect real-time data, which is processed by the microcontroller. Based on the sensor readings, necessary adjustments are made to maintain optimal storage conditions, thereby reducing spoilage and improving quality.

## 📊 Outcome
- Significant reduction in post-harvest losses  
- Improved storage efficiency and product quality  
- Cost-effective solution for farmers
